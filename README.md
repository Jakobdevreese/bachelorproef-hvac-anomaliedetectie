# 🏢 Overdraagbare Unsupervised Anomaliedetectie bij Hybride HVAC-systemen

Welkom bij de GitHub-repository van mijn bachelorproef. **Deze repository stelt alle code, data, algoritmes en Jupyter Notebooks beschikbaar die volledig open-source ontwikkeld en gebruikt zijn tijdens dit onderzoek.**

## 📖 Over het onderzoek

Dit project focust zich op het ontwikkelen van een schaalbare, datagedreven Fault Detection and Diagnostics (FDD) pipeline voor hybride HVAC-installaties (zoals geothermische warmtepompen met een BEO-veld gecombineerd met gascondensatieketels). Omdat gelabelde storingsdata in reële gebouwbeheersystemen (GBS) vrijwel nooit beschikbaar is, maakt dit onderzoek uitsluitend gebruik van *unsupervised machine learning* technieken. 

Het doel is om een pipeline te bouwen die, zonder manuele herconfiguratie of labels, het normaalgedrag van een gebouw leert en accuraat afwijkingen detecteert, waarna de code overdraagbaar is naar andere, gelijkaardige gebouwen in een patrimonium.
